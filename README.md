# Personal CV Website

## Description
This project is a personal CV website designed to present my professional background, skills, and experience in a clear and visually engaging format.

## Features
- Responsive two-column layout
- Sidebar with profile image, contact details, and skills
- Main content area with:
  - Personal profile section
  - Education
  - Work experience
  - Volunteer experience
- Custom colour palette and modern styling
- Clean, readable layout with strong visual hierarchy

## Technologies Used
- HTML
- CSS
- CSS Grid for layout

## Project Structure
- `index.html` – Main webpage
- `styles.css` – Styling and layout
- `IMG_8606.JPG` – Profile image
